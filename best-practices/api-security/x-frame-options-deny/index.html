X-Frame-Options: Deny

Send X-Frame-Options: deny header.

The X-Frame-Options header prevents the page from being displayed in an iframe, which is commonly used in clickjacking attacks. By setting the value of this header to deny, you are telling the browser not to display the page in any iframe.
